# Frontend Documentation Overview

Welcome to the ViceKanBan Frontend Documentation. This directory explains the architecture, component patterns, and state management of the application.

## Directory Structure

| File / Folder | Description |
| :--- | :--- |
| [ARCHITECTURE.md](./ARCHITECTURE.md) | Next.js App Router, Route Groups, and Middleware logic. |
| [COMPONENTS.md](./COMPONENTS.md) | UI Components and Shared Layout structure. |
| [UTILS_STATE.md](./UTILS_STATE.md) | API Fetching, Authentication, and Theme Context. |

---

## Tech Stack
- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **State/Auth**: React Context + HTTP Cookies
